# Support Threadlines

Threadlines is free and open source. The primary ways to help are reporting
reproducible bugs, improving documentation, testing releases, and contributing
focused pull requests.

## Project support

- Use GitHub Issues for reproducible bugs and focused feature requests.
- Use GitHub Discussions for usage questions and broader ideas when Discussions
  are available.
- Report security issues privately as described in [SECURITY.md](./SECURITY.md).

The project does not currently promise response times or service-level
agreements.

## Sponsorship

Sponsorship is optional financial support for continued development and project
costs, including domains, code signing, CI, and hosted Phone Link infrastructure.

A sponsorship does not purchase technical support, feature delivery, roadmap
influence, advertising, public attribution, private repository access, or any
other reward. Sponsors receive no special rights in the project, and public
recognition is not promised. Any different arrangement would require a separate
written agreement.
