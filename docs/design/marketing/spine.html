c1 · the source control panel

Git is the product, not a slash command.

A live source control panel sits beside every conversation. The working tree updates as the agent edits, so review, staging, commits, and pull requests all happen where the work happens.

  • Per-file diff stats, grouped by status and polled live every few seconds.
  • Commit, push, open a PR from one panel, with a branch per thread.
  • GitHub, GitLab, Azure DevOps, Bitbucket for pull and change requests.
  • Changes-only diffs with a slim commit graph that halos the current node.
